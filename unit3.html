Quick Notes (easy language)

  • Homogeneous coords: use (x, y, 1) so translation is a matrix.
  • Translation: moves point by (Tx, Ty). Matrix: [1 0 Tx; 0 1 Ty; 0 0 1]
  • Scaling: scales by (Sx, Sy). Matrix: [Sx 0 0; 0 Sy 0; 0 0 1]
  • Rotation: rotate by θ: [cosθ -sinθ 0; sinθ cosθ 0; 0 0 1]
  • Combined: multiply matrices. Order matters — last operation matrix is leftmost.
  • Window → Viewport: scale+translate to map world coords into pixel window.
